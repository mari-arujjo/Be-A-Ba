# Bê A Bá

O jogo Bê A Bá foi desenvolvido no **GameMaker Studio 2** utilizando **GML (GameMaker Language)**, com o objetivo de auxiliar crianças de 6 a 10 anos no processo de alfabetização de forma dinâmica, lúdica e divertida.
TCC produzido em 2023 por alunos do CEEP Lourdinha Guerra.

https://downloadbeaba.carrd.co/
